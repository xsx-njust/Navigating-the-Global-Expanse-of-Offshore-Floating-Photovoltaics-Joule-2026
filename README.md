Navigating the Global Expanse of Offshore Floating Photovoltaics
This is the public landing page for the study titled "Navigating the Global Expanse of Offshore Floating Photovoltaics".

Source Code & Data Availability
To ensure data security and comply with confidentiality agreements during the peer-review process, the source code (Scripts 01-04) and the high-fidelity dummy dataset are currently hosted in a Private Repository.

For Peer Reviewers:
If you are a reviewer assigned by Joule and wish to access the full codebase for reproducibility verification, please use one of the following methods:

Request via Editorial Office: Ask the editor to provide your GitHub username to the authors.

Direct Contact: Email the corresponding author with your GitHub username.

Once we receive your GitHub ID, we will add you as a Temporary Collaborator with read-only access to the private repository.

The full open-source version of this repository will be made public upon formal acceptance of the manuscript.
